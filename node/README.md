# dust-scripts / node

Node scripts used by PowerShell for DUST

## Setup

1. Create a `.env` file
    ```bash
    PIFU_FILE_PATH=<full-file-path-to-pifu-json>
    ```

## get-dust-pifu

Find person object and/or memberships for person from PIFU file

1. `node get-dust-pifu.json <employeeNumber>`
